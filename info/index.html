About

You can find this site's GitHub repository here.

Ultimately, I chose to put this site together using basic HTML, CSS, and vanilla JavaScript. I had debated whether it would be more useful to highlight my skills in, say, React, but what it came down to was a question of philosophy - in the end, I will almost always choose a simpler, less dependent-on-outside-resources approach if the project won't clearly benefit from those outside resources. No matter how I did it, this would be a minimally interactive project, made of mostly static parts. The basic tools are more than enough for the task.
