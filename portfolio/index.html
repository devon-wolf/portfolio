Minion Bot

Bot | Dashboard | Server

Obnoxious Discord bot delivering Minion memes in response to trigger words

I built this bot for a friend using the Discord.js library, initially hardcoded with a set of starter responses and later expanded to allow the bot admin to add and remove responses and triggers to a Node/Express/Postgres backend via a React frontend. The bot makes a fetch to the API when it first comes online, and can be manually triggered to make a fresh fetch as needed, making the bot's response time much quicker than if it needed to make an API call on every message.

Deployed on Heroku and Netlify

Cactus Bot

GitHub

Discord bot using the Merriam-Webster API to deliver thesaurus results

This bot was initially intended to a prickly proctor for writing sprints, and was the original project that inspired Shame-b0t. It was the experimentation ground for a lot of my bot and Discord.js learning, and ultimately the main functionality that came out of it was this thesaurus command.

It uses Superagent to hit the Merriam-Webster API with a search for the user-provided argument, then shapes those results into a user-friendly embed.

Deployed on Heroku

Bot Template

GitHub

Ongoing project building a clear, versatile template for new coders to build Discord bots

I began to build this template to provide a clear starting point for folks new to bots, keeping in mind the struggles I had when trying to get started. Like all the bots here, it uses the Discord.js library. There are two sets of many of the files - a commented version explaining what each line does, and an uncommented version for ease of seeing the bigger picture. This is very much a work in progress.
